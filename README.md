# Cartola FC

## Descrição

Projeto inspirado no fantasy game de futebol Cartola FC. Permite gerenciar times, jogadores e escalações.

## Tecnologias

- Backend: C# (.NET 6), Minimal API
- ORM: Entity Framework Core
- Banco de dados: SQLite
- Front-end: HTML + JavaScript

## Como executar

1. Navegue até a pasta `api` e execute:
   ```bash
   dotnet restore
   dotnet run
   ```
2. Abra o arquivo `index.html` no navegador.
